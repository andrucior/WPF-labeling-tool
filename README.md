**Labeling tool**
Tool used for photo labelling

**Features**
1. Loading files from the computer and displaying the file tree in an interactive way
2. Then the selected photos can be labeled in the next window
3. The cropped photos (with the names given by the labels) are saved in the specified library on the computer
